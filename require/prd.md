# Project overview

**LinQ**는 AI 기반의 스마트 일정 관리 서비스입니다. 사용자의 일상 패턴과 업무
스타일을 학습하여, 개인화된 일정 제안, 자동 정리, 일정 충돌 해결 등의 기능을
제공함으로써 더욱 효율적인 하루를 설계할 수 있도록 돕습니다.

LinQ는 단순히 일정을 저장하는 앱이 아니라, 사용자의 삶을 이해하고 스스로 일정을
구성하며 변화하는 환경에 능동적으로 반응하는 **AI 비서**입니다.

# 기술 스택

## Frontend (현재 구현됨)

- **Framework**: React Native 0.74.5 + Expo 51.0.0
- **Language**: TypeScript 5.3.3
- **상태 관리**: Zustand 5.0.5, React Query 5.80.7 (설치됨)
- **스타일링**: React Native StyleSheet (네이티브)
- **네비게이션**: Expo Router 3.5.24
- **개발 도구**: ESLint 9.29.0, Prettier 3.5.3

## Backend (계획됨)

- **Database**: MongoDB (주 데이터베이스)
- **Cache**: Redis (캐싱 및 세션)
- **AI Core**: Solar Pro (한국어 특화) + LangChain

## 개발 환경

- **플랫폼**: iOS & Android
- **개발 환경**: macOS (darwin 24.5.0)
- **패키지 관리**: npm

│   └── _layout.tsx        # 앱 전체 레이아웃

│   │   └── index.ts      # Event 타입 등 정의됨
